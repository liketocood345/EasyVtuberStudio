Portable training entry points for THA4/THA3 release layout.

Paths are relative to PORTABLE_ROOT (GitHub ZIP root).

Quick map:
  docs/training/run_body_train.bat        -> tools/training/run_body_train.bat
  docs/training/run_body_train_450k.bat   -> tools/training/run_body_train_450k.bat
  docs/training/package_bai_student.ps1   -> tools/training/package_student.ps1

Training outputs default to:
  workspace/distill_outputs/

Launch THA4 demo/training commands through:
  face-puppeteer-ui-enhancements-ai-code\talking-head-anime-4-demo\bin\run.bat
