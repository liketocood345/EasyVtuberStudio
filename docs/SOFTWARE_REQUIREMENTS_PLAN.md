### 1.2 产品定位（硬约束）

- **产品类型：** 直播现场临时舞台工具，聚焦临时效果与临场调整便捷性（oid **[32]**，**已完成** 写入计划）
- **不是：** 完整 3D/Live2D 创作平台、互动编排中控、完整播放器/时间线工具
- **图层实施节奏：** 三层 todo——① 五层模式 ② 无限层 ③ 图层变化与细分（oid **[59]**，**待实施**）
- **L1 当前口径（2026-05-30 确认）：** 严格五层 + 透明 PNG 素材；**不含**单色抠图/批量抠图；主 UI 为勾选「启用图层混合」后弹出的**独立六预览窗口**（非后处理栏卡片）
